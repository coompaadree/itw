Quem Somos?

   Somos um grupo de três alunos da Faculdade de Ciências de Lisboa, do curso de Tecnologias de Informação, e fomos desafiados a criar um site que permite aos seus clientes jogar o Jogo da Memória através do seu browser, para a cadeira de Introdução às Tecnologias Web.

   Utilizamos apenas o software Visual Studio Code, no qual utilizamos as linguagens Web HTML e CSS tendo ainda em vista vir a utilizar JavaScript para a realização completa do trabalho.
